Pythonanywhere-ga Upload Qilish


1) Birinchi Qadam:

Proyektingizni settings.py da

ALLOWED_HOSTS = ['*']
2) Ikkinchi Qadam:

PowerShell da proyektingizni muhitini yoqib

pip freeze > requirements.txt

Shuni Copy Qilasiz

3) Uchinchi Qadam:

Pythonanywhere-da hisob yarating - Bu yerni bosing. Ro'yxatdan o'tgandan so'ng siz sahifani shunday ko'rishingiz mumkin

4) To'rtinchi Qadam:

Endi Konsol-ni bosing, keyin Bash-ni tanlang, siz shunday ko'rasiz

5) Beshinchi Qadam:

GitHub Dan Proyekt Linkini Olish

6) Oltinchi Qadam:

Bash-da quyidagi buyruqlarni bajaring:

git clone sizning github proyekt linkingiz

cd Proyektingiz nomi/

python -m venv yangi muhit nomi #Muhit yaratish

source muhitingiz nomi/bin/activate #Muhitni activatsiya qilish

pip install -r requirements.txt #Muhit ga keraklik barcha narsani urnatadi masalan:django, pillow va hokazo

pwd

va pwd qilgandan so'ng chiqqan yo'nalishni copy qilasiz
